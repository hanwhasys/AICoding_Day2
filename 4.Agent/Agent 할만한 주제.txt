
## 난이도별 추천 (쉬운 순서)

1. **회의 요약 및 기록 Agent**
   - 텍스트 입력 → LLM → 요약/핵심 정리
   - 코드 20~30줄로 구현 가능
   - 즉시 시연 가능

2. **GPT 기반 문서/템플릿 작성 Agent**
   - 프롬프트 + 변수 입력 → 문서 생성
   - 이메일, 보고서, 공지사항 자동 작성
   - 프롬프트 엔지니어링 설명에 최적

3. **상황별 안내/헬프데스크 Agent**
   - FAQ 챗봇 형태
   - 질문 입력 → 답변 출력
   - RAG 개념 설명 가능

4. **일상 스케줄링 Agent**
   - 자연어 일정 입력 → 구조화된 일정 출력
   - JSON 형식 변환 예제로 좋음

5. **자동 뉴스 요약 및 분석 Agent**
   - URL/텍스트 입력 → 3줄 요약
   - 웹 크롤링 + LLM 조합

6. **개인 금융 비용 관리 Agent**
   - 영수증 텍스트 → 카테고리 분류 + 금액 정리
   - 간단한 데이터 처리 예제

7. **키워드 기반 글/카피 제작 Agent**
   - 키워드 입력 → 광고 문구/SNS 포스트 생성
   - 마케팅 활용 사례로 좋음

8. **이메일 자동 분류/응답 Agent**
   - 이메일 텍스트 → 카테고리 분류 → 자동 답변 생성
   - 실무 활용도 높음

9. **시험/퀴즈 자동 출제 Agent**
   - 학습 자료 입력 → 문제 생성
   - 교육 분야 활용 예제

10. **트렌드 및 시장 조사 Agent**
    - 키워드 입력 → 관련 데이터 수집/분석
    - API 연동 개념 설명 가능
